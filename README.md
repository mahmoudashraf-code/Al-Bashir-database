# what is non-relational database
A non-relational database, also known as NoSQL (which stands for "not only SQL"), is a type of database that does not use the traditional tabular relational database model that is used in relational databases like MySQL, Oracle, and SQL Server. 

In contrast to relational databases, non-relational databases are designed to handle large amounts of unstructured or semi-structured data, such as key-value pairs, documents, graphs, or columns. They are often used in applications where performance, scalability, and flexibility are key requirements.

Non-relational databases are typically used in distributed systems, cloud computing, real-time applications, and big data analytics. Some examples of non-relational databases include MongoDB, Cassandra, HBase, Couchbase, and Redis.
